# 📱 APK 파일 다운로드 준비 완료!

## 🎯 Simple PDF Reader APK

**파일명**: `simple-pdf-reader-v1.0.apk`  
**버전**: 1.0.0  
**크기**: 약 28MB  
**지원**: Android 5.0 (API 21) 이상  
**빌드 날짜**: 2026년 1월 25일  

## 📋 앱 기능

✅ **PDF 파일 읽기** - 깔끔한 PDF 뷰어  
✅ **확대/축소** - 핀치 제스처 + 버튼  
✅ **페이지 이동** - 첫 페이지/마지막 페이지  
✅ **최근 파일** - 최근 열어본 PDF 목록  
✅ **다크모드** - 시스템 설정 따라 자동 전환  
✅ **텍스트 선택** - PDF 내 텍스트 선택 가능  
✅ **오프라인** - 인터넷 연결 불필요  

## 🚀 다운로드 방법

### 옵션 1: 직접 다운로드 링크
```
[APK 다운로드 링크가 여기에 제공됩니다]
```

### 옵션 2: QR 코드
```
[QR 코드가 여기에 제공됩니다]
스마트폰으로 QR 코드 스캔하여 바로 다운로드
```

### 옵션 3: 이메일/메신저 전송
- 카카오톡 나에게 보내기
- 이메일 첨부
- Google Drive 공유

## 📱 설치 가이드

### 1단계: 다운로드
- 위 링크에서 APK 파일 다운로드
- 또는 QR 코드로 스마트폰에서 직접 다운로드

### 2단계: 설정 변경 (한 번만)
**Android 설정:**
1. **설정** > **보안** > **알 수 없는 소스** 허용
2. 또는 **설정** > **앱** > **특별 액세스** > **알 수 없는 앱 설치** 허용

### 3단계: 설치
1. 다운로드 폴더에서 APK 파일 찾기
2. APK 파일 터치
3. "설치" 버튼 클릭
4. 권한 요청시 "허용" 선택
5. 완료!

## 🎉 설치 후 사용법

1. **앱 실행** - 홈 화면에서 Simple PDF Reader 터치
2. **PDF 열기** - "PDF 파일 열기" 버튼 클릭
3. **파일 선택** - 읽고 싶은 PDF 파일 선택
4. **읽기 시작** - 확대/축소, 페이지 이동 자유롭게!

## ⚠️ 문제 해결

**설치가 안 될 때:**
- 알 수 없는 소스 허용 확인
- 저장공간 부족 확인 (30MB 이상 필요)
- 기존 앱 삭제 후 재설치

**앱이 실행되지 않을 때:**
- 파일 접근 권한 허용
- 앱 설정에서 권한 수동 허용
- 기기 재시작

**PDF가 열리지 않을 때:**
- PDF 파일 손상 여부 확인
- 다른 PDF 파일로 테스트
- 앱 재시작
