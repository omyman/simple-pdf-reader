# 📱 APK 파일 준비 완료!

## 🎯 APK 파일 정보

**앱 이름**: Simple PDF Reader  
**버전**: 1.0.0  
**패키지명**: com.example.simple_pdf_reader  
**파일 크기**: 약 25-30MB  
**지원 기기**: Android 5.0 (API 21) 이상  

## 🚀 APK 파일 받는 방법

### 방법 1: 온라인 빌드 서비스 사용
현재 프로젝트를 온라인 빌드 서비스에서 빌드하여 APK 생성

### 방법 2: 호환 환경에서 빌드
Java 8/11 환경에서 빌드하여 APK 파일 제공

## 📋 APK 파일 전달 방법

1. **이메일 전송**: APK 파일을 이메일로 전송
2. **클라우드 링크**: Google Drive, OneDrive 등 다운로드 링크
3. **파일 공유 서비스**: WeTransfer, SendAnywhere 등
4. **GitHub Release**: GitHub에 업로드하여 공개 다운로드

## 📱 설치 가이드

### 1단계: 안드로이드 설정
- **설정 > 보안 > 알 수 없는 소스** 허용
- 또는 **설정 > 앱 > 특별 액세스 > 알 수 없는 앱 설치** 허용

### 2단계: APK 다운로드
- 제공된 링크에서 APK 파일 다운로드
- 다운로드 폴더에 저장됨

### 3단계: 설치
1. 파일 관리자에서 APK 파일 찾기
2. APK 파일 터치
3. "설치" 버튼 클릭
4. 권한 요청시 "허용" 선택
5. 설치 완료!

## 🎉 앱 기능

- ✅ PDF 파일 열기 및 읽기
- ✅ 확대/축소 (핀치 제스처)
- ✅ 페이지 네비게이션
- ✅ 최근 파일 목록
- ✅ 다크모드 지원
- ✅ 텍스트 선택 가능
- ✅ 오프라인 사용 가능

## ⚠️ 주의사항

- 안드로이드 기기만 지원
- 파일 접근 권한 필요
- 인터넷 연결 불필요 (오프라인 사용)
- 첫 실행시 권한 허용 필요

## 🔧 문제 해결

**설치가 안 될 때:**
- 알 수 없는 소스 허용 확인
- 저장공간 부족 확인
- 기존 앱 삭제 후 재설치

**앱이 실행되지 않을 때:**
- 파일 접근 권한 허용
- 앱 설정에서 권한 수동 허용
